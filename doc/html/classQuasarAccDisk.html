Detailed Description

Class used to process quasar radiation template and add it to calculated jet radiation; at the moment quasar radiation is not stricty calculated but a radius-loud quasar radiation template from Elvis et al. (1994) is used. Class has some basic methods to calculate standard multitemperature Shakura-Sunyayev disk radiation but it is not used currently as it need much, much detailed study and modelling of dusty torus, coronae etc
